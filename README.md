## Installation

```bash
conda create -n stride_new python=3.9 -y
conda activate stride_new
pip install uv
uv pip install -r requirements.txt
```

## Tune Hyperparameters
```bash
python experiments/tune_stride.py --task pen --trials 30 --device cuda
python experiments/tune_stride.py --task hammer --trials 30 --device cuda
python experiments/tune_stride.py --task relocate --trials 30 --device cuda
python experiments/tune_stride.py --task door --trials 30 --device cuda
```

## Train Tuned Model

Run the benchmark for each task using the tuned STRIDE config generated by `tune_stride.py`:
```bash
python experiments/benchmark.py --task pen --device cuda --num-trials 50 --stride-params results/tune_pen_best_params.json
python experiments/benchmark.py --task hammer --device cuda --num-trials 50 --stride-params results/tune_hammer_best_params.json
python experiments/benchmark.py --task relocate --device cuda --num-trials 50 --stride-params results/tune_relocate_best_params.json
python experiments/benchmark.py --task door --device cuda --num-trials 50 --stride-params results/tune_door_best_params.json
```
